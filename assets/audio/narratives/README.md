# Narrative Audio Files

This directory contains audio narrations for literary synonyms' multi-part narratives.

## Audio Generation

Audio files are generated using Microsoft Edge TTS with the following voices:

### departir (Literary Argentinian voices)
- **Part 1**: `ar_female_1` (Argentinian Spanish, female)
- **Part 2**: `ar_male_1` (Argentinian Spanish, male)
- **Part 3**: `ar_female_1` (Argentinian Spanish, female)

## File Structure

```
departir_part_1.mp3  - Sal√≥n literario scene (ar_female_1)
departir_part_2.mp3  - Tertulia development (ar_male_1)
departir_part_3.mp3  - Meta-reflection (ar_female_1)
```

## Text Content

Full text for each part is available in `data/audio_metadata.json` under the `narratives` section.

## Voice Selection Rationale

Argentine Spanish voices were chosen for "departir" to match the literary, formal register of the narrative and reflect the cultural sophistication of Latin American intellectual traditions (tertulias, literary salons).
